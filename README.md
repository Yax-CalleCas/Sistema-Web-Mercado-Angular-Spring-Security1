#  Sistema de Gestión de Socios y Alquiler de Puestos

![Java](https://img.shields.io/badge/Java-17-blue)
![Spring Boot](https://img.shields.io/badge/SpringBoot-3.x-green)
![Angular](https://img.shields.io/badge/Angular-Frontend-red)

![Status](https://img.shields.io/badge/Status-Active-success)

Sistema web full-stack para la gestión de socios, alquiler de puestos y control de pagos.  
Desarrollado con **Spring Boot + Angular + JWT** siguiendo arquitectura REST.

---

##  Características

-  Autenticación con JWT
-  Gestión de socios (CRUD completo)
- Catálogo de puestos disponibles
-  Registro de alquileres
-  Control de pagos y recibos
-  Generación de comprobantes
-  Interfaz moderna con Angular + Bootstrap
-  API REST optimizada con Spring Boot

---

## Tecnologías usadas

### Backend
- Java 17
- Spring Boot
- Spring Security + JWT
- JPA / Hibernate
- MySQL

### Frontend
- Angular
- TypeScript
- Bootstrap 5
- RxJS
